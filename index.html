MBTA Mapper

Passion project

View MBTA vehicles in real time!

View the repo! View the site!
  • Uses requests, pandas and SQLAlchemy (python) to generate and insert both static and real-time data into a SQLite database
  • Renders with leaflet, a JavaScript library for interactive maps and has some html and css to style it
  • I made it after I realized I was dependent on Keolis's internal realtime map for my commute and wanted to make something that I could use, also this site was really the only place to view the MBTA like this, but it looks like it's straight outta 1998.
