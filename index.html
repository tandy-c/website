MBTA Mapper

Passion project

Realtime MBTA mapping @mbtamapper.com.

View the repo! View the site!
  • There were really no websites (this site doesn't count because it's bad) that showed the MBTA in real time on a map. I became dependant on Keolis' internal map for my commute, but I lost access to it after my co-op ended and thus I made it to satisfy myself.
  • Uses requests, pandas and SQLAlchemy (Python) to generate and insert both static and real-time data into a SQLite database.
  • The data is sent using Flask (Python), and rendered with leaflet, a JavaScript library for interactive maps and has some html and css to style it.
  • There's a preview of the map below for the commuter rail, but there are other routes available here!
