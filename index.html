MBTA Mapper

Passion project

Realtime MBTA mapping @mbtamapper.com.

View the repo! View the site!

I wanted to make something that both me and my friends could use on a daily basis after I lost access to Keolis's internal map app, especially since there weren't any good looking/usable MBTA realtime maps. This website is the only one I know of and it looks and feels like something from 1998.

Implementation:

  • This project uses requests, pandas and SQLAlchemy (Python) to pull and dump static and realtime data into a SQLite database.
  • After some processing, Flask (Python) sends the data to leaflet (JavaScript), which renders it. Additional syling is done with HTML and CSS.
  • I containerized it with Docker and deployed it to Azure.
  • My web app constantly evolves; I try to update it whenever I have time and ideas for new features or fixes.

A preview of the web app for the commuter rail, but there are other routes available here. Note that other route types might take longer to load.
